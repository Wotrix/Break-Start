## [1.0.1] - 2026-02-26

### 🐛 缺陷修复 (Bug Fixes)
- 修复了因为动画防碰撞安全锁(`easterEggActive`)导致的“彩蛋触发次数被静默吞咽”的 Bug。现在，在当前正在播放某个彩蛋时，其余触发词仅会被拦截但保留生命周期，直到动画播完仍可正常触发。

### ✨ 新增功能 (Features)
- **彩蛋系统 (Easter Eggs)**
  - 增加了触发隐藏关卡的词库拦截机制，且设定为**单页加载周期内限触发一次**，防止动画疲劳干扰正常搜索。
  - 补充双语触发机制，现在所有隐藏彩蛋同时支持中文与英文触发！
  - 新增 `SNOW` / `下雪` 飘雪彩蛋：屏幕顶端洒落柔和的雪花。
  - 新增 `SHAKE` / `地震` 摇晃彩蛋：界面引发一阵剧烈震动。
  - 新增 `NEON` / `霓虹` 炫丽彩蛋：搜索框瞬间切换为赛博朋克风的霓虹灯效。
  - 保留 `WT💗LM` 专属浪漫彩蛋 (开发者首字母纪念)：全屏樱花/爱心/字母上浮飘落动效。
  - 保留 `MATRIX` / `黑客帝国` 数字雨彩蛋：全屏绿色代码流数字雨。
  - 保留 `DO A BARREL ROLL` / `大翻滚` 谷歌经典大回旋彩蛋：页面主体顺时针 360 度翻转。
  - 保留 `COIN FLIP` / `抛硬币` 幸运彩蛋：页面弹出 3D 硬币抛掷动画，并在联想框内直接给出正反面预测结果。

- **搜索体验飞跃**
  - **搜索建议联想框交互重构**：抛弃原有的方块融合式（变形突兀问题），改为类似 macOS Spotlight / Raycast 的**独立悬浮玻璃卡片 (Detached Floating Panel)**。带有自然的 8px 空气感留白间距，向下丝滑出现。
  - **无焦点预览 (Hover to Show)**：现在只要搜索框内有文字，无需点击聚焦，鼠标滑过搜索区域即可像抽屉一样查看历史/联想内容。
  - **搜索框 UI 优化**：现在激活状态也永久保持完美的胶囊形状。
  - 搜索联想结果框子项大小适配缩小，更加灵动自然。

### 🎨 界面与设计优化 (UI/UX)
- **全局分辨率与视距调整**：微调了全局网格、卡片以及组件大小，缩小了字体字重，使得界面更显“高分辨率”的精致现代高保真质感。
- **毛玻璃与光影**：全站应用更细腻的 `backdrop-filter` 磨砂质感和自发光立体阴影。
- 移除了搜索框激活时产生的不必要紫色光效，回归统一纯净的白色系果冻玻璃风。
